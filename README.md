# el-facon-frontend
Frontend del sistema de reservas del restaurante "El Facon" desarrollado con React.

## Descripcion
Este proyecto consiste en el desarrollo de una aplicación web para la gestión de reservas de mesas en un restaurante.

Los usuarios tras registrarse e iniciar sesión para realizar las reservas en fechas y horarios especificos.
Los administradores tendrán acceso a un panel donde podrán gestionar los usuarios y las reservas realizadas en el sistema.

### Funcionalidades principales
- Registro de usuarios
- Inicio de seseión
- Creación de reservas
- Visualización de reservas
- Panel de administración
- Gestión de usuarios
- Gestión de reservas

### Tecnologías
- React
- React Router
- Bootstrap

### Instalacion 
- Clonar el repositorio:
  git clone https://github.com/Jogutgon/el-facon-frontend.git

- Instalar dependencias:
  npm install

- Ejecutar el proyecto en modo desarrollo:
  npm run dev

### Conexión con el backend
El frontend consume una API REST desarrollada con Node.js y Express.

URL de la API:
  http://localhost:5173/
